# -java数据结构预算法
##1.选择排序
```
思路：循环不变量：[0,i)是排好序的，[i,n)是未排序的
	每次都从i到n中选择一个最小的数放在i的位置
	时间复杂度为O(n^2)
```
##2.插入排序
```
思路：循环不变量：[0,i)是排好序的，[i,n)是未排序的
	每次让i的位置和[0,i)的数据进行比较，将i位置的数插入到[0,i)中合适的位置。
	时间复杂度为O(n^2)
	插入排序和选择排序的区别在于：插入排序每次不会去遍历[0,i)的数据，而选择排序，每次遍历的是[i,n)的数据
	
	优化：插入排序在比较i和[0,i]过程中，如果i位置的值小于[0,i)，每次比较都会进行一次数组交换位置，通过记录i位置的临时变量t，将i插入的位置到i的值平移，最后将插入i的位置赋值为临时变量t,进而避免数组交换位置的开销。
```
